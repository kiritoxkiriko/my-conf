# Raycast Template

This file records the non-sensitive parts of the current local Raycast setup.
The binary `raycast_cong.rayconfig` export may contain private workflow data, so
review it before replacing or sharing it.

## Window and Navigation

- Global hotkey monitoring: enabled.
- Global hotkey: configured locally, value omitted.
- Window mode: default.
- Navigation command style: macOS.
- Follow system appearance: enabled.
- Keep window visible when focus is lost: disabled.
- Favorites in compact mode: enabled.

## Quicklinks

- Auto-fill link: disabled.
- Quick search: disabled.

## File Access

Folder read permissions are enabled for common user folders, cloud storage, and
removable volumes. Specific local absolute paths are omitted from this template.

## Extensions

- Weather menubar command record exists locally.
- Other extension records are app-managed and may include private metadata.

## Local Data

- Main Raycast application data is stored in encrypted app databases.
- Emoji data is stored separately in a SQLite database.
- Analytics/update/cache folders are app-managed and should not be committed as
  source templates.
